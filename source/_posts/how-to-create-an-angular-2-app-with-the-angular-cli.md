## How to create an Angular 2 app with the [angular-cli](https://cli.angular.io/)

### Run these commands

``` bash
npm install -g angular-cli
ng new my-awesome-app
cd my-awesome-app

```

### Run server

``` bash
ng serve
```

### Build for production

``` bash
ng build --prod
```

More info: [angular-cli](https://cli.angular.io/)

Enjoy!
